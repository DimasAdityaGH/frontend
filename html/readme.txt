apa itu html?
-html singakatan dari hypertext markup language
-html adalah bahasa markup standar untuk membuat web
-html menggambarkan struktur halaman web
-html terdiri dari serangkaian elemen
-elemen html memeberi tahu browser cara menampilkan konten
-elemen html memeberi potongan konten seperti judul, paragraf dan  tautan

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

-penjelasan struktur html diatas:
<!DOCTYPE html> = mendefinisikan dokumen ini adalah html5
<html> = elemen elemen akar halaman html
<head> = elemen berisi informasi meta tentang html
<title> = elemen untuk menentukan judul untuk halaman html
<body> = elemen mendefinisikan tubuh dokumen dan merupakan wadah bagi paragraf, heading, hyperlink, image, tabel, judul dll
<h1> = mendefinisikan judul besar
<p> - mendefinisikan paragraf


Apa itu elemen html?
elemen html didefinisikan oleh tag awal, beberapa konten, dan tag akhir. contoh:
<tagname>konten<tagname>

ada beberapa elemen html tidak memiliki konten seperti ( <br> ) elemen ini sebut elemen kosong dan elemen kosong tidak mempunyai tag akhir

attribut html:  
-semua elemen html dapat memiliki attribut
-attribut memeberikan informasi tambahan tentang elemen
-attribut selalu ditentukan ditag awal
-attribut biasanya datang dalam pasangan nama/nilai seperti: nama="nilai"
Ringkasan Bab attribut
Semua elemen HTML dapat memiliki atribut
The hrefatribut <a>Menentukan URL halaman link pergi ke
The srcatribut <img>menentukan path ke gambar yang akan ditampilkan
The widthdan heightatribut <img>memberikan informasi ukuran untuk gambar
The altatribut <img>menyediakan teks alternatif untuk gambar
The styleatribut digunakan untuk menambah gaya untuk elemen, seperti warna, font, ukuran, dan lebih
The langatribut dari <html>tag menyatakan bahasa halaman Web
The titleatribut mendefinisikan beberapa informasi tambahan tentang elemen
